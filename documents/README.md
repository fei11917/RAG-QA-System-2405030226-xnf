# 文档存放目录

请将您的PDF或DOCX格式的文档放入此文件夹。

## 支持的文档格式

- PDF文件 (.pdf)
- Word文档 (.docx)

## 文档要求

1. 文档内容应与"自然语言处理技术"相关
2. 建议至少准备5份文档
3. 文档大小建议不超过50MB
4. 确保文档内容清晰可读

## 使用方法

### 方法1：使用Web界面上传
1. 启动应用后，在左侧边栏点击"上传文档"
2. 选择PDF或DOCX文件
3. 点击"构建知识库"按钮

### 方法2：使用命令行版本
1. 将文档放入此文件夹
2. 运行 `python rag_cli.py`
3. 系统会自动读取并构建知识库

## 示例文档主题

建议收集以下主题的文档：
- 自然语言处理基础
- 词向量与语言模型
- 深度学习在NLP中的应用
- Transformer模型
- 文本分类与情感分析
- 命名实体识别
- 机器翻译
- 问答系统
